Paragraph Tags

Paragraph tags are used to indicate paragraphs on the page.

Bold and Strong Tags

The bold tag renders text in bold but does not suggest emphasis of the content.

The strong tag also renders text in bold but the text is emphasized.

I and EM tags

The i tag renders the text in italics but does not suggest emphasis of the content.

The em tag renders the text in italics but the text is emphasized.

Images

Image Tags

The image tag allows developers to add images to the web page. The image must have src and alt attributes, src is the path of the image. The alt attribute is a description for those who may need more accessibility.
